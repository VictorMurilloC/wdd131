Site Plan: Rafting Adventure

Site Name

Rafting Adventure

This site provides an engaging platform to explore thrilling whitewater rafting experiences and book trips.

Site Purpose

The purpose of this website is to offer a seamless booking experience for whitewater rafting enthusiasts, provide safety guidelines, and showcase exciting rafting trips.

Scenarios

  • What safety precautions should I take before going rafting?
  • How can I book a rafting trip online?

Color Schema

The following colors will be used throughout the website:

  • Dark Green (#3e5c57): Used for headings and navigation background.
  • Light Teal (#53b5a0): Used for accents and call-to-action buttons.
  • Light Yellow (#d9c26f): Used for backgrounds.
  • Light Peach (#f7d3a1): Used for section highlights.
  • Light Gray (#9ba7c0): Used for body text and subtle borders.

Typography

The following fonts will be used across the website:

  • Georgia: Used for headings.
  • Roboto: Used for body text for readability.
